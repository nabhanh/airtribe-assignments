# Airtribe Task Manager REST API Project

## Description

This is a REST API project for a task manager application. It is built with Node and Express.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## The following endpoints are available

```bash
$ GET /tasks
$ GET /tasks/priority/:priority
$ GET /tasks/:id
$ POST /tasks
$ PUT /tasks/:id
$ DELETE /tasks/:id
```
